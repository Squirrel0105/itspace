Тема 1: Основные типы данных в JavaScript

В JavaScript существует несколько типов данных,
которые помогают разработчикам работать с различной информацией.
Давайте разберем самые важные типы:
числа, строки, логические значения, массивы и объекты.

1. Число (Number)

Числа используются для хранения любых числовых значений: целых и дробных.

Пример:
let age = 30; // Целое число
let price = 19.99; // Дробное число
let price = 19.99; // Дробное число
let total = age + price; // Сложение чисел
console.log(total); // 49.99

Где используется?

Числа применяются для расчетов, например,
для подсчета возраста, цены товаров, или выполнения математических операций.

2. Строка (String)

Строки представляют текстовую информацию и заключаются в кавычки (" ", ' ', или `` `).

Пример:
let name = "Alice";
let greeting = "Привет!";
let message = greeting + " Меня зовут " + name + ".";
console.log(message); // Привет! Меня зовут Alice.

Где используется?

Строки нужны для хранения имен, сообщений, текстовых данных и общения с пользователем.

3. Булево значение (Boolean)

Булевы значения принимают только два значения: true (истина) или false (ложь).
Они помогают принимать решения в коде.

Пример:
let isOnline = true; // Пользователь онлайн
let hasAccess = false; // Доступа нет
if (isOnline && hasAccess) {
console.log("Доступ разрешен");
} else {
console.log("Доступ запрещен");
}
// Результат: Доступ запрещен

Где используется?

Булевы значения часто применяются для проверок условий и управления потоком выполнения программы.

4. Массив (Array)

Массивы — это упорядоченные списки данных.
Они могут содержать числа, строки, другие массивы и даже объекты.

Пример:
let fruits = ["Яблоко", "Банан", "Апельсин"];
console.log(fruits[0]); // "Яблоко" (первый элемент)
fruits.push("Груша"); // Добавляем новый элемент
console.log(fruits); // ["Яблоко", "Банан", "Апельсин", "Груша"]

Где используется?

Массивы полезны для хранения списков, например, продуктов, задач или пользователей.

4. Объект (Object)

Объект — это сложная структура данных, которая хранит информацию в формате "ключ: значение".

Пример:
let user = {
name: "Alice",
age: 25,
isOnline: true,
};
console.log(user.name); // "Alice"
console.log(user["age"]); // 25
// Добавление новых свойств
user.city = "Москва";
console.log(user); // { name: "Alice", age: 25, isOnline: true, city: "Москва" }

Где используется?

Объекты применяются для группировки связанных данных, например,
информации о пользователе, настройках или товарах.


Итоги

  • 1. Number — для чисел (возраст, цена, результаты расчетов).
  • 2. String — для текстов (имена, сообщения, описания).
  • 3. Boolean — для логических условий (включено/выключено, истина/ложь).
  • 4. Array — для списков данных (списки фруктов, задач).
  • 5. Object — для сложных структур (данные о пользователе, настройки).
