Тема 2: Переменные в JavaScript:var, let, const

В JavaScript переменные — это "контейнеры" для хранения данных.
Вы можете использовать три способа их создания: var, let и const.
Разберем их просто и понятно.

1. var — старый способ (лучше не использовать)

  • Работает по всему коду или внутри функции, игнорируя блоки.
  • Можно переопределять.

Пример:
var name = "Alice";
if (true) {
var name = "Bob"; // Переменная переопределится
}
 console.log(name); // "Bob"

Проблема: переменная меняется за пределами блока.

2. let — современный и безопасный

  • Работает только в пределах блока, где объявлена.
  • Можно менять значение.

Пример:
let score = 10;
if (true) {
let score = 20; // Это другая переменная внутри блока
}
console.log(score); // "10"

3. const — для неизменяемых данных

  • Значение нельзя переназначить.
  • Работает в пределах блока.

Пример:
const maxLives = 3;
maxLives = 5; // Ошибка!

Что использовать?

  • let — для переменных, которые меняются.
  • const — для данных, которые остаются неизменными.
  • var — лучше избегать.
