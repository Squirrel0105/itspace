Тема 3: Методы работы с массивами в JavaScript

Массивы — это упорядоченные списки данных, с которыми мы часто работаем в JavaScript.
Для удобного управления элементами массива существуют специальные методы.
Рассмотрим основные из них: push, shift, forEach, filter и join.

1. push — добавляет элемент в конец массива

Метод push добавляет один или несколько элементов в конец массива.
Он изменяет исходный массив и возвращает новую длину массива.

Пример:
let fruits = ["Яблоко", "Банан"];
fruits.push("Апельсин"); console.log(fruits); // ["Яблоко", "Банан", "Апельсин"]

Где использовать?

Добавление новых данных, например, товаров в корзину.

2. shift — удаляет первый элемент массива

Метод shift удаляет первый элемент массива и возвращает его значение.
Исходный массив изменяется.

Пример:
let fruits = ["Яблоко", "Банан", "Апельсин"];
let firstFruit = fruits.shift(); console.log(firstFruit); // "Яблоко"
console.log(fruits); // ["Банан", "Апельсин"]

Где использовать?

Удаление старых данных, например, первого элемента в очереди.

3. forEach — выполняет функцию для каждого элемента

Метод forEach перебирает массив и выполняет заданную функцию для каждого элемента.
Он ничего не возвращает.

Пример:
let fruits = ["Яблоко", "Банан", "Апельсин"];
fruits.forEach((fruit) => {
console.log(`Я люблю ${fruit}`);
});
 // Вывод:
// Я люблю Яблоко
// Я люблю Банан // Я люблю Апельсин

Где использовать?

Обход массива для выполнения операций с каждым элементом.

4. filter — создает новый массив с отфильтрованными элементами

Метод filter возвращает новый массив, содержащий только те элементы,
которые соответствуют заданному условию.

Пример:
let numbers = [1, 2, 3, 4, 5];
let evenNumbers = numbers.filter((num) => num % 2 === 0); console.log(evenNumbers); // [2, 4]

Где использовать?

Фильтрация данных, например, отбор товаров по цене или категории.

5. join — объединяет элементы массива в строку

Метод join преобразует массив в строку,
разделяя элементы указанным символом (по умолчанию — запятая).

Пример:
let fruits = ["Яблоко", "Банан", "Апельсин"];
let fruitString = fruits.join(", "); console.log(fruitString); // "Яблоко, Банан, Апельсин"

Где использовать?

Создание строк из массива, например, списка покупок или тегов.

Итоги

  • 1. push — добавляет элемент в конец массива.
  • 2. shift — удаляет первый элемент массива.
  • 3. forEach — выполняет действие для каждого элемента.
  • 4. filter — создает новый массив с элементами, которые соответствуют условию.
  • 5. join — объединяет элементы массива в строку.
