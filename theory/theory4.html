Тема 4: Работа с DOM в JavaScript: Основные методы

DOM (Document Object Model) — это структура, которая представляет HTML-документ как дерево объектов.
JavaScript позволяет вам взаимодействовать с элементами на странице с помощью различных методов.
Рассмотрим основные:document.getElementById, document.querySelector, addEventListener, textContent и document.createElement.

1. document.getElementById — поиск элемента по ID

Метод getElementById позволяет найти элемент на странице по его уникальному атрибуту id.

Пример:
let header = document.getElementById("main-header");
console.log(header); // Привет, мир!
console.log(fruits); // ["Яблоко", "Банан", "Апельсин"]

Где использовать?

Быстрый доступ к конкретному элементу с заданным id, например, заголовку или блоку.

2. document.querySelector — универсальный поиск

Метод querySelector ищет первый элемент, соответствующий CSS-селектору.

Пример:
let firstParagraph = document.querySelector(".text");
console.log(firstParagraph); // Это текстовый блок

Где использовать?

Для поиска элементов по тегам, классам, ID или сложным селекторам.

3. addEventListener — добавление обработчика события

Метод addEventListener добавляет обработчик события к элементу (например, клики, ввод текста, наведение мыши).

Пример:
let button = document.querySelector(".btn");
button.addEventListener("click", () => {
console.log("Кнопка нажата!");
});

Где использовать?

Для взаимодействия с пользователем, например, кликов на кнопки или отправки форм.

4. textContent — изменение текста элемента

Свойство textContent позволяет получить или изменить текст внутри элемента.

Пример:
let header = document.getElementById("main-header");
header.textContent = "Добро пожаловать!"; console.log(header.textContent); // "Добро пожаловать!"

Где использовать?

Для обновления текста на странице, например, вывода сообщений пользователю.

5. document.createElement — создание нового элемента

Метод createElement создает новый элемент, который можно добавить на страницу.

Пример:
let newDiv = document.createElement("div");
newDiv.textContent = "Это новый блок!";
document.body.appendChild(newDiv); // Добавляем блок в конец страницы

Где использовать?

Для динамического добавления контента, например, новых сообщений, карточек или кнопок.

Итоги

  • 1. document.getElementById — найти элемент по его id.
  • 2. document.querySelector — найти элемент с помощью CSS-селектора.
  • 3. addEventListener — реагировать на события (например, клики).
  • 4. textContent — изменить текст элемента.
  • 5. document.createElement — создать новый элемент.
